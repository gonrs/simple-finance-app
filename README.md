# app
<div style="background-color: #f2f2f2; padding: 10px;">
  <h3>To start a server:</h3>
  <p>Create a .env file with variables like this</p>
  <ul>
    <li>DB_HOST</li>
    <li>DB_PORT</li>
    <li>DB_USERNAME</li>
    <li>DB_PASSWORD</li>
    <li>DB_NAME</li>
    <li>JWT_SECRET</li>
  </ul>
  <h4>Then run it:</h3>
  <ul>
    <li>cd server</li>
    <li>npm run start:dev</li>
  </ul>
  <h3>To start a client:</h3>
  <p>Create a .env file with variables like this</p>
  <ul>
    <li>VITE_URL (server url like http://localhost:8000/api/ )</li>
  </ul>
  <h4>Then run it:</h3>
  <ul>
    <li>cd client</li>
    <li>npm i </li>
    <li>npm run dev</li>
  </ul>
  <h3>
    List of technologies used :
  </h3>
  <h5>Server - </h5>
  <ul>
    <li>Node.js</li>
    <li>Nest js</li>
    <li>PostgreSQL</li>
    <li>JWT auth</li>
    <li>TypeScript</li>
  </ul>
  <h5>Client - </h5>
  <ul>
    <li>React</li>
    <li>Redux</li>
    <li>Vite</li>
    <li>TypeScript</li>
  </ul>
</div>
