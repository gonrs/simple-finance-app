# app
<div style="background-color: #f2f2f2; padding: 10px;">
  <h3>To start a server:</h3>
  <p>Create a .env file with variables like this</p>
  <ul>
    <li>DB_HOST</li>
    <li>DB_PORT</li>
    <li>DB_USERNAME</li>
    <li>DB_PASSWORD</li>
    <li>DB_NAME</li>
    <li>JWT_SECRET</li>
  </ul>
  <h4>Then run it:</h3>
  <ul>
    <li>cd server</li>
    <li>npm run start:dev</li>
  </ul>
  <h3>To start a client:</h3>
  <p>Create a .env file with variables like this</p>
  <ul>
    <li>VITE_URL</li>
  </ul>
  <h4>Then run it:</h3>
  <ul>
    <li>cd client</li>
    <li>npm i </li>
    <li>npm run dev</li>
  </ul>
</div>
